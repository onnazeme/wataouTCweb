　☆基本操作☆
 　請使用下列的其中一種按鍵。

　[Z鍵] [Enter] [空白鍵]…

　[方向鍵] … 選擇指令、選項
  [X鍵][Esc] … 取消

　使用PC時，也能用滑鼠操作。
